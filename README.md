# comp323_week5_ahernandez42

How to run:
- Press Space to start
- Using numbers 1,2,3,4 to modify graphics
- Esc to Exit

Controls:
- Using Arrow Keys or WASD to move

What I added 
- Screen Flash when hit
- Text on screen when hit
- wiggling arms animation when moving

CoreLoop:
- Collect all the coins to move to the next level
- If hit and lose all three lifes game over
